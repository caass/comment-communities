About this project

Comment Communities visualizes the comment activity on reddit in real time

by repeatedly poking the reddit comments API for new comments, sorting them by subreddit, and using D3 to visualize the subreddits as bubbles.

The colors are chosen randomly from a D3 palette, and the sizes are proportional to the number of comments left in a subreddit.

All of the data is kept in the front-end, so you can look at it yourself by opening your javascript console and looking at the subreddits global object.

We have a full README.md in our GitHub Repository, linked below.

Reddit Comment Communities

A project by the GT Big Data Club to visualize the activity on Reddit in real time.

A lot of people on Reddit have a lot of opinions, and they aren't afraid to express them in the form of comments. Constantly. By checking all the new comment activity on Reddit, we can group incoming comments by their Subreddit and visualize the activity in real time.
